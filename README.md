# 大學專題
應用JSP建立一個動態的健康照護網站平台。

## 開發工具與技術
- 系統分析：UML
- 開發工具：Eclipse
- 繪圖軟體：Illustrator、Photoshop
- 伺服器：Apache Tomcat
- 資料庫：Mysql
- 程式語言：JSP、HTML、CSS、JavaScript

## 負責工作
- 系統文件撰寫：使用個案圖、流程圖、活動圖
- 資料庫：資料表設計、資料庫建立
- 介面設計：負責大部分的介面設計
- 系統開發：負責50%以上的功能模組開發
- 系統整合：彙整組員開發的功能模組
